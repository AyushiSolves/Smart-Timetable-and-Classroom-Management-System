# Smart-Timetable-and-Classroom-Management-System
A web-based Classroom Management System with Teacher and Student panels. Enables timetable management, classroom booking, requests, and notices. Built using HTML, CSS, and JavaScript with localStorage for dynamic data handling.

The system consists of two main panels: Teacher Panel and Student Panel.

🔹 Teacher Panel Features:
- Dashboard with summary of classes, requests, and notices
- Timetable management (add, view, delete schedules)
- Classroom booking with conflict detection
- Request system (approve/reject classroom requests)
- Notices management (create and view announcements)
- Profile management with editable details

🔹 Student Panel Features:
- Dashboard showing today's classes and notices
- View full timetable
- View notices
- Profile section with logout functionality

🔹 Key Concepts Used:
- LocalStorage for data persistence (simulating backend)
- Dynamic UI updates using JavaScript
- Role-based navigation (Teacher / Student)
- Basic conflict detection logic
- Responsive and clean UI design

🔹 Future Improvements:
- Integration with real backend (Node.js / Database)
- Authentication system with secure login
- Real-time updates
- Mobile responsiveness

This project demonstrates the implementation of a real-world academic management system with focus on usability, logic building, and frontend development.
